Major machine learning techniques

  • Regression/Estimation:

    • It's used for predicting a continuos value, for example, the price of a house based on its characteristics

  • Clasification:

    • It's used for predicting a class or a category of a case, for example, if a cell is bening or maling

  • Clustering:

    • It's used for clustering groups of similar cases, for example, find similar patients

  • Association:

    • It's used for finding items or events that often co-occur, for example, grocery items that are usually bought together

  • Anomaly Detection:

    • It's used for discovering abnormal and unsual cases. for example, credit card fraud detection

  • Sequence Mining:

    • It's used for predicting next events, for example, the click stream in web sites

  • Dimension Reduction:

    • It's used for reducing the size of data

  • Recommendation Systems:

    • It's used for associate different tastes or preferences of persons and then recommend new items they might like

Supervised vs Unsupervised learning

Supervised learning

We observe and direct the execution of a task, project or activity. We'll be supervising a machine learnign model by teaching it by loading it with necessary knowledge, so we can have a predict on the future. We train the model with some data from a labeled dataset. It's important to note that the dataset is labeled and the enviroment is very controlled by us. There are two kinds of supervising models: CLASSIFICATION AND REGRESSION

Unsupervised learning

We don't supervise the model, but we let the model work on his own to discover information that is not visible for the human eye. Since we know little information about the data or the outcomes that are to be expected, it has more difficult algorithms. It's important to note that here we deal with unlabeled dataset of information and our task is to classify or to cluster it. By the way, the enviroment we work on is very less controlled. The most widely used techniques are: DIMENSION REDUCTION, DENSITY ESTIMATION, MARKET BASKET ANALYSIS AND CLUSTERING

Regression

It has two kind of variables: Dependent and independent variable. The dependent variable is the goal we try to predict, and the independent variable (can be one or more) is the cause of the dependent variable. Basiclly there are two types of regression models:

Simple Regression

It's when ONE independent variable is used to estimate one dependent variable. It can be:

  • Simple Linear Regression
  • Simple Non-Linear Regression

Multiple Regression

It's when ONE OR MORE independents variables are used to estimate one dependent variable. It can be:

  • Multiple Linear Regression
  • Multiple Non-Linear Regression

Model Evaluation in Regression Models

The most widely used evaluation models are:

  • Train and Test on the same data set

    • Due to test with training data, it produces a high training accuracy but this is not necessarily a good thing. It may be over-fit and a non-generalized model By the way this produces a low out-of-sample accuracy

  • Train and Test Split

    • Due to test with non-training data, it produces a higher out-of-sample accuracy wich is what we want. The problem with this model is a high dependency on wich datasets the data is trained and tested.

  • K-fold Cross-validation

    • It performs multiple Train and Test split evaluations and average the results. Then the total result is a more consistent out-of-sample accuracy

Evaluation Metrics in Regression Models

In the regression model, an error consists on the meassure of how far the data points are from the trend line generated by the algorithm.

The most widely used evaluation metrics are:

  • Mean Absolute Error
  • Mean Squared Error
  • Root Mean Squared Error
  • Relative Absolute Error
  • Relative Squared Error
  • R^2 = 1 - RSE

    • This is not an error but is a popular metric for the accuracy of the model. The higher R-squared, the higher the model fits the data
